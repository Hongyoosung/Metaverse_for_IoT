# WSL에서 DID_Wallet 빌드

## 프로그램 설치
Visual Studio Code와 WSL(Ubuntu 20.04.6 LTS)를 설치한 뒤, 명령어를 사용하여 연동

ubuntu를 실행하여 다음의 명령어들을 입력

    $ sudo apt-get install build-essential gdb
    $ mkdir projects
    $ cd projects
    $ mkdir helloworld
    $ cd helloworld
    $  code .
